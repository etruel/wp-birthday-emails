== Description ==

WP Birthday emails is a plugin that add a birthday field to users and automatically sends a happy birthday email to each user on his birthdays. 

You can use this plugin to make your users or customers happier and increase their loyalty.

Then in user's profile page will have a new input for birthday. You need to add user birthday manually.

To compose the email, please go to **WP Birthday emails** menu.  Use the tags to use user fields in the email body.

The plugin was taken and modified from happy emails and is made and maintained by [netmdp](http://www.netmdp.com)

== Installation ==

1. Go to *Plugins* and search for **WP Birthday emails**
2. Click **Install** button to install the plugin
3. Click **Activate** to activate the plugin

Then in user's profile page will have a new input for birthday. You need to add user birthday manually.

To compose the email, please go to **WP Birthday emails** menu.
